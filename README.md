Twitter
=======

Get Twitter tweets to create custom timeline and facebook tabs.

$("#yourDiv").twitter({ twitterID : "yedincikatajans", // Your twitter id. tweetCount : 0}, // Number of tweets to display ( 0 = All tweets ) function(){ // callback });
